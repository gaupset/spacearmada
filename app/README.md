# App

Project containing source code for the Space Armada game.

**Note!** The project is a work-in-progress revamp of the original `android` project. The goal is
to incrementally rebuild the `android` project here using ECS instead of MVC. This is an
alternative to refactoring the entire `android` project which fixes configuration issues. When this
project is finished, it should replace the old `android` project.

The project is a [libGDX](https://libgdx.com/) project generated with
[gdx-liftoff](https://github.com/libgdx/gdx-liftoff).
